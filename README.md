✈️ Análisis de Lealtad y Comportamiento de Clientes: Aerolínea
📋 Descripción del Proyecto
Este proyecto realiza un análisis integral de los datos de un programa de lealtad de una aerolínea. El objetivo es entender el perfil demográfico de los clientes, sus hábitos de vuelo y cómo factores como el nivel educativo o el estado civil influyen en métricas clave como el Salario y el CLV (Customer Lifetime Value).

El proyecto se divide en tres fases críticas: Limpieza y Exploración, Análisis Estadístico Descriptivo y Visualización de Datos.

🛠️ Estructura del Proyecto
El repositorio está organizado de la siguiente manera:

Fase_1_exploracion_y_limpieza.ipynb: Unión de datasets, gestión de nulos (imputación por mediana), corrección de tipos de datos y eliminación de registros inconsistentes (salarios negativos).

Fase_2_analisis_estadistico.ipynb: Cálculo de estadísticos descriptivos (media, mediana, moda, desviación estándar) y análisis de distribución de variables clave.

Fase_3_visualizacion_Datos.ipynb: Creación de dashboards visuales, análisis de correlación entre distancia y puntos, y segmentación demográfica.

src/: Carpeta de código fuente que contiene funciones de soporte (soporte.py) para mantener los notebooks limpios y modulares.

📊 Insights Principales
Tras el análisis, se han extraído las siguientes conclusiones de negocio:

Perfil del Cliente: El segmento mayoritario está compuesto por personas casadas (Married), con un balance de género casi perfecto (50/50).

Correlación Puntos-Distancia: Se identificaron 5 niveles de acumulación de puntos. Sorprendentemente, la tarjeta de lealtad no es el único factor, sugiriendo que la clase de tarifa influye significativamente.

Análisis Salarial: Existe una relación directa entre el nivel educativo y el salario promedio, siendo los perfiles con Doctorado y Máster los de mayor poder adquisitivo.

Limpieza de Datos: Se detectaron y corrigieron anomalías críticas, como salarios negativos y nulos en el campo de ingresos, asegurando la integridad del análisis.
